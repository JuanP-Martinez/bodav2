# Boda Web Juanjo y Loli

Proyecto Vite + React listo para subir a GitHub y publicar en Vercel.

## Requisitos

- Node.js instalado
- Cuenta de GitHub
- Cuenta de Vercel

## Arrancar en local

```bash
npm install
npm run dev
```

## Compilar

```bash
npm run build
```

## Publicar en Vercel

1. Sube esta carpeta a un repositorio de GitHub.
2. Entra en Vercel.
3. Pulsa **Add New > Project**.
4. Importa el repositorio.
5. Pulsa **Deploy**.

Vercel detectará Vite automáticamente.
